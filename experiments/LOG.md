# Experiment Log

Narrative record of the experiment loop. One entry per experiment, newest
first. Format:

```
## exp_NNN_name — KEEP/REVERT (+0.00XX ensemble val AUC)
Date. One paragraph: what was tried, why, what happened, what it implies.
Serve-time computability (for new features): one sentence.
```

Numbers live in `results.jsonl`; this file is for reasoning. Negative
results get entries too — they are how the next person avoids repeating
them.
